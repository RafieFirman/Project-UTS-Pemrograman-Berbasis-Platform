Project Saya

Healer

Healer adalah aplikasi manajemen rumah sakit yang dibuat menggunakan konsep OOP pada bahasa Python, dengan framework Kivy & KivyMD untuk tampilan GUI, dan JSON untuk penyimpanan data. Aplikasi ini memiliki fitur seperti pendaftaran pasien, penjadwalan janji temu, dan manajemen data dokter.

UniFind

UniFind adalah aplikasi pencarian universitas di Indonesia yang dibuat menggunakan bahasa C++ dengan penerapan konsep Struktur Data. Aplikasi ini memungkinkan pengguna mencari universitas berdasarkan lokasi, jurusan, dan akreditasi serta menampilkan informasi lengkap dari setiap universitas.

Music Box Berbasis Arduino Nano

Music box ini dibuat menggunakan Arduino Nano sebagai mikrokontroler utama, dengan bahasa C untuk memutar melodi "Happy Birthday To You". Dirakit sepenuh hati sebagai kado ulang tahun untuk teman.
